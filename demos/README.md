# Demos

这里放自己实现的小实验，不直接放课程官方作业答案。

当前 demo：

```text
demos/
├─ cpp-starter/        # 纯 C++ 图形学基础模板
├─ meshlib-starter/    # MeshLib 网格处理基础模板
├─ transformation-viewer/
├─ triangle-rasterizer/
└─ ray-sphere-intersection/
```

每个 demo 目录建议包含：

- `README.md`：说明目标、运行方法和结果
- `src/`：源码
- `results/`：运行截图或输出图像
